# Product Overview

This is a multi-tenant marketplace platform connecting customers, vendors, and delivery partners. The system enables:

- **Customers**: Browse products from multiple vendors, place orders, track deliveries, manage addresses and profiles
- **Vendors**: Manage products, accept/reject orders, set operating hours, handle order fulfillment
- **Delivery Partners**: Accept delivery assignments, update delivery status, manage service areas
- **Super Admins**: Oversee platform operations, manage users, vendors, orders, and service areas

## Core Features

- Multi-role authentication (password and OTP-based)
- Real-time order management with status tracking
- Shopping cart with vendor-specific carts
- Payment processing with multiple methods (Card, UPI, Net Banking, COD)
- Service area-based vendor and delivery partner assignment
- Notification system for order updates
- Audit logging for administrative actions
- Rate limiting and security features

## Business Model

The platform operates as a marketplace where vendors list products, customers place orders, and delivery partners fulfill deliveries within defined service areas. Each entity has role-specific dashboards and workflows.
